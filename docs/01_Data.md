# Data Description

## Satellite Data
- Mission: Sentinel-1A
- Mode: IW (Interferometric Wide Swath)
- Product Type: SLC
- Polarization: VV&VH
- Orbit Direction: Descending

# Acquisition Details
- Temporal Coverage: July 2022 to March 2025
- Revisit Time - 12 Days
- Look Direction - Right Looking

# Study Area
- Location: Bengaluru, India
- Geographic Extent: Urban and peri_urban regions
- Terrains: Mostly flat with dense build-up areas

# Auxillary Data
- DEM: SRTM 30 m
- Land Mask: Global Land Mask

# Data Source
- Provider: Alaska Satellite Facility (ASF) and Copernicus Platform
- Access Method: ASF API via PyGMTSAR or ASF API in Python
